# 命理分析保存目录

这个目录用于存储您保存的命理分析结果。

## 文件格式

- 所有分析结果以 Markdown (.md) 格式保存
- 文件名格式：`命理分析_八字_日期.md`
- 例如：`命理分析_甲子_乙丑_丙寅_丁卯_2024-08-23.md`

## 文件内容

每个保存的文件包含：
- 分析的八字信息
- 分析时间
- 完整的AI分析结果

## 查看方式

您可以使用以下方式查看保存的分析：
1. 使用任何文本编辑器打开
2. 使用 Markdown 编辑器查看（推荐）
3. 直接在文件浏览器中预览

## 注意事项

- 文件会永久保存在本地
- 可以随时删除不需要的分析结果
- 建议定期备份重要的分析结果
